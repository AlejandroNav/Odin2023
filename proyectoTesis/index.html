Programación web

La programación web es el conjunto de técnicas y habilidades usadas para desarrollar aplicaciones y sitios web interactivos. Implica la escritura de código utilizando diferentes lenguajes de programación, como HTML, CSS y JavaScript, para crear la estructura, el diseño y la funcionalidad de las páginas web.

HTML se usa para definir la estructura y el contenido de una página web. Se emplea para ocasionar elementos como encabezados, párrafos, listas, tablas, enlaces y otros elementos de contenido.

CSS se utiliza para controlar el aspecto visual y el diseño de una página web. Permite aplicar estilos, como colores, fuentes, márgenes y tamaños, a los elementos HTML, lo que ayuda a producir diseños atractivos y coherentes en toda la aplicación web.

JavaScript es un lenguaje de programación de alto nivel que se emplea para agregar interactividad y funcionalidad dinámica a una página web. Con JavaScript, es posible realizar acciones como validación de formularios, manipulación del contenido de la página, animaciones, interacción con el usuario y comunicación con servidores.

El Internet

Internet es una red internacional de ordenadores conectados. Ninguna empresa es propietaria de Internet; se trata de un esfuerzo cooperativo regido por un sistema de normas y reglas. El objetivo de conectar ordenadores es, por supuesto, compartir información.

Hay muchas formas de transmitir información entre ordenadores, como el correo electrónico (POP3/IMAP/SMTP), la transferencia de archivos (FTP), el shell seguro (SSH) y muchos otros modos especializados sobre los que se construye Internet. Estos métodos estandarizados para transferir datos o documentos a través de una red se conocen como protocolos.

HTTP es el conjunto de reglas que define como sucede esta comunicación, técnicamente es el protocolo (TCP/IP) base que permite que los clientes (navegador web) y los servidores web se comuniquen entre sí para solicitar y enviar recursos, como páginas web, imágenes, videos y otros archivos. HTTP se basa en el modelo cliente-servidor, donde el cliente (generalmente un navegador web) realiza una solicitud a un servidor y espera una respuesta.

La solicitud del cliente incluye un método (como GET, POST, PUT, DELETE) que indica el tipo de acción que desea realizar, así como la URL del recurso solicitado. Cuando el servidor recibe la solicitud, procesa la solicitud y envía una respuesta al cliente. La respuesta del servidor incluye un código de estado que indica si la solicitud fue exitosa (como 200 OK, o 404 no existe).

Páginas Web

Una página web es un documento electrónico que se encuentra en el Internet y que puede ser accedido mediante un navegador web. Es la unidad básica de información en línea y está compuesta por contenido multimedia, como texto, imágenes, videos y otros elementos interactivos. Para representar páginas web se emplea un navegador web (Chrome, IE, Edge Firefox, etc.) La función principal de un navegador es presentar el recurso web elegido, solicitándolo al servidor y mostrándolo en la ventana del navegador.

El recurso suele ser un documento HTML, pero también puede ser un PDF, una imagen u otro tipo de contenido. El usuario especifica la ubicación del recurso mediante un URI (Identificador Uniforme de Recursos). Para las páginas web el documento principal es el archivo HTML, pero también suele incluir archivos CSS para estilos y JS para funcionalidad en el lenguaje JavaScript, además de archivos adicionales.

Una página web puede ser estática o dinámica. Una página web estática muestra el mismo contenido a todos los usuarios y no cambia con el tiempo, mientras que una página web dinámica puede adaptarse al usuario, mostrar contenido personalizado y actualizarse en tiempo real mediante la interacción con una base de datos u otros servicios vía lenguaje de programación JavaScript en el front-end y diversas opciones de lenguajes(JS, Java, Python) para la funcionalidad en back-end.

Las páginas web están organizadas en sitios web (archivos HTML), que son colecciones de páginas web interconectadas mediante enlaces. Los sitios web pueden tener una estructura jerárquica, con una página de inicio (Index) que sirve como punto de entrada principal, y varias páginas secundarias organizadas en directorios o secciones.
